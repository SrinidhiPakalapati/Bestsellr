What is Bestsellr?

Bestsellr is a YSWS where you write a short 1–2 page story, build a cool webpage for it using HTML & CSS, and create your own downloadable e-book so others can actually read it.Ship your project and get cool book themed prizes.

FAQ's Section

Who can participate in Bestsellr?

Anyone aged 13-18 (inclusive) can participate in Bestsellr. No need of any experience.

If I am new to programming, can I get any guidence?

Yep! You can get a guide made exclusively for this workshop where we will be making a sample webpage to make you fimiliar with basic html and css.

Can I use AI to write my story?

Nope! You could not use AI generated stories. We only encourage human written creative stories.

Do I need to record my progress using Hackatime??

Yep!You need to also submit a Hackatime or Hack Club Lapse for getting proof that you have done everything right!
