What is Bestsellr?

Bestsellr is a YSWS where you write a short 1–2 page story, build a cool webpage for it using HTML & CSS, and create your own downloadable e-book so others can actually read it. Ship your project and get cool book-themed prizes.

FAQ's

Who can participate in Bestsellr?

Anyone aged 13–18 (inclusive) can participate. No prior experience is required.

If I am new to programming, can I get guidance?

Yep! You will get beginner-friendly guidance where we build a sample webpage together using basic HTML and CSS.

Can I use AI to write my story?

Nope! We encourage completely human-written, original, and creative stories.

Do I need to record progress using Hackatime?

Yep! You should submit Hackatime or Hack Club Lapse recorded project to verify your work.
